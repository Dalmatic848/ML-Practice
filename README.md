# Machine Learning 

Репозиторий содержит практические работы по основным направлениям машинного обучения:  
классификация, регрессия, кластеризация и набор данных.

## Содержание

### Classification
- KNN, Random Forest, SVM
- Метрики качества: accuracy, precision, recall, F1, ARI
- Визуализация результатов

### Regression
- Линейная, Ridge, Lasso регрессия
- Отбор признаков и оценка модели (MSE, R²)

### lusterization
- K-Means, DBSCAN, иерархическая кластеризация

- Анализ распределения кластеров и сравнение с реальными метками

### data
- Наборы данных в формате `.csv`
- Содержат числовые и категориальные признаки
- Готовы к использованию в ноутбуках
